# Pahari Patrika

Hindi news website built with Next.js 16 and WordPress headless CMS.

## Features

- Next.js 16 with App Router
- WordPress Headless CMS with WPGraphQL
- Hindi language support
- SEO optimized with Yoast integration
- Google News sitemap
- AdSense native ads
- ISR caching for performance

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm start
```

## Environment Variables

Required:
- `WP_GRAPHQL_ENDPOINT` - WordPress GraphQL endpoint
- `NEXT_PUBLIC_SITE_URL` - Production site URL
- `NEXT_PUBLIC_ADSENSE_CLIENT_ID` - Google AdSense client ID
