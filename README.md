# Next.js + WordPress (GraphQL) — Clean News Template

- Hero + news list on homepage
- Category menu in mobile header
- Pages appear in the footer (no separate index)
- Article & pages resolved by URI/slug (no 404)
- Category pages use slug → IDs (schema-safe)
- Site logo pulled from WordPress media (search: "logo")
- Localhost friendly, Next.js 16-compatible

## Run
```
cp .env.example .env.local
npm i
npm run dev
```
Visit http://localhost:3000
