# Social Media Embeds - Quick Start

## How to Add Embeds in WordPress

### YouTube Videos

Simply add a YouTube block in WordPress:

1. Click the "+" button in the editor
2. Type "YouTube"
3. Select "YouTube" block
4. Paste your video URL:
   - `https://www.youtube.com/watch?v=dQw4w9WgXcQ`
   - `https://youtu.be/dQw4w9WgXcQ`
   - `https://www.youtube.com/shorts/VIDEO_ID`

The video will automatically embed with a responsive player!

### Twitter/X Posts

Add a Twitter block:

1. Click "+" in the editor
2. Type "Twitter" or "X"
3. Select "Twitter" block
4. Paste your tweet URL:
   - `https://twitter.com/username/status/1234567890`
   - `https://x.com/username/status/1234567890`

The tweet will automatically embed with dark theme styling!

## Features

✅ **Automatic Processing** - Just paste the URL, everything else is automatic
✅ **Responsive Design** - Works perfectly on mobile and desktop
✅ **Modern Styling** - Rounded corners, proper spacing, dark theme
✅ **Fast Loading** - Scripts load asynchronously, no performance impact
✅ **SEO Friendly** - Search engines can still crawl the original links
